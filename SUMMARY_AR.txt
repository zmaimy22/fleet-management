🚀 كيفية الاستخدام
═══════════════════════════════════════════════════════════════════════════

الخطوة 1: تشغيل الخادم
   $ cd server
   $ npm install
   $ node server.js

الخطوة 2: تشغيل الواجهة (في نافذة أخرى)
   $ npm install
   $ npm run dev

الخطوة 3: فتح المتصفح
   → http://localhost:3000

الخطوة 4: استيراد البيانات
   1. سجل الدخول (admin / admin123)
   2. اذهب إلى الصفحة الرئيسية
   3. اختر "📤 Import Excel"
   4. حمّل "Calendario_Noviembre_2024.xlsx"
   5. انتظر اكتمال المعالجة
